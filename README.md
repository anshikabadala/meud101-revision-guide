## 🌟 Project Overview

The **MEUD101 Complete Revision Guide** is a single-page interactive study application designed to help engineering students master the essential modules of Engineering Drawing, Mechanical Engineering, and Civil Engineering. It features high-quality revision cards, 20 fully explained MCQs (covering the 2025 exam paper and 2026 predictions), key formulas, and exam strategy guides.

---

## ⚡ Key Features

* **Interactive Navigation:** Seamless tab switching allows students to bounce between sections (MCQs, Drawing, Mechanical, Civil, Formulas, and Exam Strategy) instantaneously.
* **Collapsible Revision Cards:** Beautifully styled cards with smooth toggle animations keep content highly readable and well-organized.
* **Complete MCQ Explanations:** 20 critical MCQs with highlighted answers, conceptual callouts, and warning banners for common pitfalls.
* **Premium Theme & Micro-animations:** Modern dark slate and vibrant blue aesthetics with subtle transitions and hover effects.
* **Responsive Layout:** Works flawlessly on desktop monitors, tablets, and smartphones.
* **100% Offline-Capable:** Built as a single, lightweight compiled file with embedded vector graphics (SVGs), ensuring extremely fast load speeds without bulky frameworks.

## 📖 Topics Covered

### ✏️ Part A: Engineering Drawing
* **Fundamentals:** Bureau of Indian Standards (BIS) SP:46 codes, line conventions, and sheet sizes.
* **Engineering Scales:** RF calculations, plain scales, and diagonal scales.
* **Engineering Curves:** Methods of drawing ellipses (Concentric Circle Method), conics classification (ellipse, parabola, hyperbola), cycloids, involutes, and spirals.
* **Projections:** First Angle vs Third Angle projections (comparisons and symbols).
* **Isometric Views:** Isometric scales, axes, lines, and drawing techniques.

### ⚙️ Part B: Mechanical Engineering
* **Thermodynamics:** Core terms defined (Cp, Cv, internal energy, enthalpy, entropy).
* **Gas Laws:** Zeroth Law, First Law ($Q = W + \Delta U$), Boyle's Law, Charles's Law, and Perfect Gas equations.
* **Steam Generators:** Cochran boiler construction, shell specs, and complete boiler classifications.
* **Engine Comparisons:** Detailed 4-mark comparisons for **SI vs CI**, **IC vs EC**, and **2-Stroke vs 4-Stroke** engines.
* **Refrigeration:** Vapour Compression Refrigeration (VCR) cycle, COP formulas, and refrigerant properties.

### 🏗️ Part C: Civil Engineering
* **Building Components:** Detailed breakdown of substructures (foundations, DPC) and superstructures (lintels, chajjas, parapets, columns).
* **Planning Principles:** All 9 planning rules (Aspect, Prospect, Privacy, Grouping, Roominess, Elegance, Circulation, Flexibility, Sanitation).
* **Structures:** Extensive comparison between Load-Bearing and RCC Framed structures.
* **Surveying:** Chain surveying, tape composition (Invar tape details), and core principles of surveying.

## 🛠️ How to Use the Revision Guide

1. Visit the [Live Website](https://anshikabadala.github.io/meud101-revision-guide/).
2. Use the top navigation bar to select the category you want to study.
3. Click on any card header (e.g., **"conic sections"** or **"SI vs CI"**) to slide open the comprehensive notes, formula boxes, and comparison tables.
4. Open the **"All Formulas"** tab for a quick, consolidated reference guide right before the exam.

## 💻 How to Edit and Update Locally

Since this project has been structured as a clean Git repository, you can easily modify the guide and push updates live.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anshikabadala/meud101-revision-guide.git
   cd meud101-revision-guide
   ```
2. **Edit the source code:**
   * Open `index.html` in your favorite code editor (like VS Code).
   * Modify the questions, add custom styling, or update links.
3. **Commit and push updates:**
   ```bash
   git add index.html
   git commit -m "Update revision guide content"
   git push
   ```
4. **Deploy:** GitHub Pages will automatically detect the push and deploy your changes within 60 seconds.
